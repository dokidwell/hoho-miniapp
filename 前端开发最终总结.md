## 🚀 下一步操作

### 1. 准备图标（你负责）

**必须准备的图标**：
- TabBar图标（10个）：
  - `tab-works.png` / `tab-works-active.png`
  - `tab-create.png` / `tab-create-active.png`
  - `tab-exchange.png` / `tab-exchange-active.png`
  - `tab-ecology.png` / `tab-ecology-active.png`
  - `tab-profile.png` / `tab-profile-active.png`

**规格**：81×81px，PNG，透明背景

**放置位置**：`frontend/static/icons/`

**推荐工具**：
- iconfont.cn（免费图标库）
- Figma / Sketch
- 或找设计师（200-500元）

### 2. 修改配置

**修改API地址** (`frontend/api/config.js`)：
```javascript
export const API_BASE_URL = 'https://api.你的域名.com'
```

**修改AppID** (`frontend/manifest.json`)：
```json
{
  "mp-weixin": {
    "appid": "你的小程序AppID"
  }
}
```

### 3. 部署后端

**使用一键部署脚本**：
```bash
ssh root@你的服务器IP
cd /opt
git clone https://github.com/dokidwell/hoho-miniapp.git
cd hoho-miniapp/backend
sudo bash deploy.sh
```

**配置HTTPS**：
```bash
apt install -y certbot python3-certbot-nginx
certbot --nginx -d api.你的域名.com
```

### 4. 编译前端

**使用HBuilderX**：
1. 下载HBuilderX
2. 导入 `frontend` 目录
3. 修改API地址和AppID
4. 运行到微信开发者工具

**使用微信开发者工具**：
1. 下载微信开发者工具
2. 导入 `frontend` 目录
3. 填入AppID
4. 编译预览
5. 上传代码

### 5. 提交审核

1. 登录小程序管理后台
2. 配置服务器域名
3. 提交审核
4. 等待审核（1-7天）
5. 审核通过后发布
