# HOHO数字藏品集换小程序

## 项目简介

HOHO是一个基于Go后端和Uni-app前端的跨平台数字藏品集换社区小程序。用户可以在这个平台上进行藏品的铸造、空投、集换等操作，所有交易都使用站内积分驱动，完全不涉及真实货币支付。

### 核心特性

- **跨平台支持**：支持微信小程序、支付宝小程序、抖音小程序等多个平台
- **积分驱动经济**：8位小数精度的站内积分系统，不可提现、不可转账
- **透明公示账本**：所有关键事件（铸造、空投、交易）都被记录并公示，实现"伪链上浏览器"效果
- **模块化架构**：核心功能独立运作，鲸探API等第三方集成可选
- **OKX风格UI**：专业、科技感十足的用户界面
- **完整后台管理**：用户管理、藏品管理、交易管理、积分管理、数据分析等

### 技术栈

| 层级 | 技术 | 说明 |
| :--- | :--- | :--- |
| **后端** | Go 1.21+ | 高性能、高并发的后端服务 |
| **前端** | Uni-app + Vue 3 | 一套代码多平台部署 |
| **数据库** | MySQL 8.0+ | 关系型数据库 |
| **缓存** | Redis | 用于积分交易的原子性操作 |
| **文件存储** | 对象存储(COS/OSS) | 藏品图片、视频等资源存储 |

## 快速开始

### 后端开发

```bash
cd backend
go mod download
go run main.go
```

### 前端开发

```bash
cd frontend
npm install
npm run dev
```
