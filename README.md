# HOHO数字作品集换小程序

> 一个基于区块链的数字作品创作、集换和社区平台

## 📖 项目简介

HOHO是一个创新的数字作品集换平台，用户可以创作、收藏、交易数字作品，并通过第三方平台（如鲸探、Waveup）映射积分，构建完整的数字作品生态系统。

### 核心特性

- **作品创作**：用户可以创作并铸造独特的数字作品
- **集换交易**：基于积分的作品交易系统
- **第三方关联**：支持鲸探、Waveup等平台作品映射
- **透明公示**：所有交易记录上链，完全透明
- **积分体系**：完整的积分获取和使用机制
- **社区生态**：白皮书、用户协议、社区活动

## 🏗️ 技术架构

### 后端技术栈

- **语言**：Go 1.20+
- **框架**：Gin Web Framework
- **数据库**：MySQL 8.0
- **缓存**：Redis 7.0
- **认证**：JWT Token
- **部署**：Docker / 直接部署

### 前端技术栈

- **框架**：Uni-app
- **语言**：Vue 3 + Composition API
- **状态管理**：Pinia
- **平台**：微信小程序
- **UI风格**：极简白色主题

## 🚀 快速开始

### 后端部署

1. **克隆仓库**
```bash
git clone https://github.com/dokidwell/hoho-miniapp.git
cd hoho-miniapp/backend
```

2. **一键部署**（推荐）
```bash
sudo bash deploy.sh
```

3. **手动部署**
```bash
# 安装依赖
go mod download

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件

# 运行
go run main.go
```

详细说明：[后端部署指南.md](./后端部署指南.md)

### 前端编译

1. **安装HBuilderX**
   - 下载：https://www.dcloud.io/hbuilderx.html

2. **打开项目**
   - 用HBuilderX打开 `frontend` 目录

3. **配置**
   - 修改 `api/config.js` 中的API地址
   - 修改 `manifest.json` 中的AppID

4. **运行**
   - 运行 → 运行到小程序模拟器 → 微信开发者工具

详细说明：[前端编译和文案修改指南.md](./前端编译和文案修改指南.md)
