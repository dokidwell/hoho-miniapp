# 小程序图标准备和放置指南

## 📋 需要准备的图标

### TabBar图标（10个，必须）

| 图标名称 | 文件名 | 尺寸 | 说明 |
|---------|--------|------|------|
| 作品-未选中 | `tab-works.png` | 81×81px | 灰色 |
| 作品-选中 | `tab-works-active.png` | 81×81px | 黑色/彩色 |
| 创作-未选中 | `tab-create.png` | 81×81px | 灰色 |
| 创作-选中 | `tab-create-active.png` | 81×81px | 黑色/彩色 |
| 集换-未选中 | `tab-exchange.png` | 81×81px | 灰色 |
| 集换-选中 | `tab-exchange-active.png` | 81×81px | 黑色/彩色 |
| 生态-未选中 | `tab-ecology.png` | 81×81px | 灰色 |
| 生态-选中 | `tab-ecology-active.png` | 81×81px | 黑色/彩色 |
| 我的-未选中 | `tab-profile.png` | 81×81px | 灰色 |
| 我的-选中 | `tab-profile-active.png` | 81×81px | 黑色/彩色 |

**规格要求**：
- 格式：PNG
- 背景：透明
- 颜色：未选中用灰色(#999999)，选中用黑色(#000000)或品牌色

## 📁 图标放置位置

### 方法1：直接放到项目里（推荐）

把准备好的图标文件放到这个目录：
```
frontend/static/icons/
├── tab-works.png
├── tab-works-active.png
├── tab-create.png
├── tab-create-active.png
├── tab-exchange.png
├── tab-exchange-active.png
├── tab-ecology.png
├── tab-ecology-active.png
├── tab-profile.png
└── tab-profile-active.png
```

**具体操作**：
1. 从GitHub下载项目代码到本地
2. 找到 `frontend/static/icons/` 目录
3. 把10个图标文件复制进去
4. 重新提交到GitHub（或者直接用这个版本编译）

## 🎨 如何快速制作图标

### 方案1：使用在线图标库（最快）

#### iconfont.cn（阿里巴巴图标库）
1. 访问 https://www.iconfont.cn
2. 搜索关键词：
   - "作品" / "gallery" / "image"
   - "创作" / "create" / "add"
   - "交换" / "exchange" / "swap"
   - "生态" / "ecology" / "leaf"
   - "我的" / "profile" / "user"
3. 选择喜欢的图标
4. 下载PNG格式（81×81px）
5. 用Photoshop或在线工具调整颜色

#### Flaticon.com
1. 访问 https://www.flaticon.com
2. 搜索图标
3. 下载PNG格式
4. 调整尺寸和颜色

### 方案2：使用Figma/Sketch设计

如果你会用设计工具：
1. 创建81×81px画布
2. 设计图标（简洁、清晰）
3. 导出PNG（2x分辨率）
4. 分别导出灰色和黑色版本

### 方案3：找设计师（最省心）

在以下平台找设计师：
- 猪八戒网
- Fiverr
- 淘宝

**参考价格**：200-500元（10个图标）

## 📝 修改TabBar配置

如果准备好了图标，需要修改 `pages.json`：

### 当前配置（使用自定义TabBar组件）
```json
{
  "tabBar": {
    "custom": true,
    "list": [...]
  }
}
```

### 如果要使用原生TabBar（有图标后）
编辑 `frontend/pages.json`：

```json
{
  "tabBar": {
    "custom": false,
    "color": "#999999",
    "selectedColor": "#000000",
    "backgroundColor": "#ffffff",
    "borderStyle": "black",
    "list": [
      {
        "pagePath": "pages/index/index",
        "iconPath": "static/icons/tab-works.png",
        "selectedIconPath": "static/icons/tab-works-active.png",
        "text": "作品"
      },
      {
        "pagePath": "pages/create/index",
        "iconPath": "static/icons/tab-create.png",
        "selectedIconPath": "static/icons/tab-create-active.png",
        "text": "创作"
      },
      {
        "pagePath": "pages/jijhuan/index",
        "iconPath": "static/icons/tab-exchange.png",
        "selectedIconPath": "static/icons/tab-exchange-active.png",
        "text": "集换"
      },
      {
        "pagePath": "pages/ecology/index",
        "iconPath": "static/icons/tab-ecology.png",
        "selectedIconPath": "static/icons/tab-ecology-active.png",
        "text": "生态"
      },
      {
        "pagePath": "pages/profile/index",
        "iconPath": "static/icons/tab-profile.png",
        "selectedIconPath": "static/icons/tab-profile-active.png",
        "text": "我的"
      }
    ]
  }
}
```

**注意**：
- 如果使用原生TabBar，需要删除页面中的 `<TabBar>` 组件
- 如果继续使用自定义TabBar，可以在组件中使用图标
