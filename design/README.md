## 🎨 设计风格

- **主题**：白色/浅色主题
- **风格**：极简科技风，参考OKX和Apple
- **字体**：简洁现代，数字使用等宽字体
- **间距**：适中，保持呼吸感
- **圆角**：统一使用圆角设计

## 📝 使用说明

1. 设计稿仅供参考，实际开发时可根据技术实现进行调整
2. 部分页面可能需要增加或修改功能
3. 颜色、字体、间距等细节以开发文档为准
4. 如有疑问，请参考 `/docs/FRONTEND_DEVELOPMENT_GUIDE.md`

## ⚠️ 注意事项

- 设计稿中的文字、图片仅为示意，实际内容以后端API返回为准
- 部分功能可能在设计稿中未体现，需根据需求文档补充
- 积分显示必须精确到8位小数
- 所有交互需考虑加载状态、错误状态、空状态
