# HOHO Park 管理后台

## 概述

HOHO Park管理后台是一个基于纯HTML/CSS/JavaScript开发的轻量级后台管理系统，用于管理平台的核心业务功能。

## 功能模块

### 1. 数据概览
- 平台核心数据统计（用户数、作品数、待审核数、今日交易）
- 最近活动记录
- 数据可视化展示

### 2. 创作审核
- 查看待审核的创作作品列表
- 审核通过/拒绝操作
- 查看作品详情和创作者信息
- 审核历史记录

### 3. 任务管理
- 添加/编辑/删除任务
- 配置任务奖励积分
- 启用/禁用任务
- 查看任务完成统计

### 4. 公告管理
- 发布/编辑/删除公告
- 置顶公告管理
- 公告发布时间设置
- 查看公告阅读统计

### 5. 阳光账户
- 平台收入透明展示
- 交易记录查询
- 收支统计分析
- 数据导出功能

### 6. 系统配置
- 分成比例配置（创作者/平台）
- 交易手续费率配置
- 功能开关管理（创作、任务、第三方插件）
- 其他系统参数配置

## 技术栈

- **前端框架**: 纯HTML/CSS/JavaScript（无框架依赖）
- **UI风格**: 简洁扁平化设计，参考Ant Design风格
- **API通信**: Fetch API
- **数据存储**: LocalStorage（token存储）

## 部署方式

### 方式一：静态文件部署

将`admin`目录下的所有文件部署到Web服务器即可：

```bash
# Nginx配置示例
server {
    listen 80;
    server_name admin.hohopark.com;
    
    root /path/to/hoho-miniapp/admin;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
}
```

### 方式二：与后端集成

将管理后台作为后端的静态资源目录：

```go
// main.go
r.Static("/admin", "./admin")
```

访问地址：`https://api.hohopark.com/admin`

## API接口

所有API接口定义在`api/config.js`中，包括：

- 认证接口：登录、登出
- 创作审核接口：列表、详情、审核
- 任务管理接口：CRUD操作
- 公告管理接口：CRUD操作
- 阳光账户接口：统计、交易记录
- 系统配置接口：获取、更新

## 使用说明

### 登录

默认访问`index.html`会检查登录状态，未登录会跳转到登录页面。

### 权限控制

所有API请求都需要在Header中携带JWT Token：

```
Authorization: Bearer <token>
```

### 数据刷新

各页面数据在切换时自动加载，也可手动刷新。
